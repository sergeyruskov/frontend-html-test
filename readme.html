Blur

Совсем простое и самое топорное решение, это загрузить одну картинку разбитую на две части, например такую, и при hover менять background-position. Недостатки: большая нагрузка на сервер и клиента в силу больших размеров картинки, сложно менять картинки. Плюсы: работает везде.

Более современная вещь, это CSS3 св-во filter, с параметром blur, очень плохо поддерживается браузерами, до сих пор находится в стадии разработки у микрософта, поэтому лучше комбинировать специальный фильтр для IE и CSS3 св-во. Например так:

	Для FF в HTML понадобится добавить SVG filter

	<svg version="1.1" xmlns="http://www.w3.org/2000/svg">

		<filter id="blur">

			<feGaussianBlur stdDeviation="3" />

		</filter>

	</svg>

	<style>
		img.blur {
			filter: blur(3px); /*Мечты, мечты*/
			-webkit-filter: blur(3px);
			-moz-filter: blur(3px);
			-o-filter: blur(3px);
			-ms-filter: blur(3px);
			filter: url(blur.svg#blur); /*FF*/
			filter:progid:DXImageTransform.Microsoft.Blur(PixelRadius='3'); /*для ветеранов*/
			zoom: 1;
		}
	</style>
Если рассчитывать на браузеры в будущем, то можно оставлять filter: blur(3px);
